# 文通Ai质检平台

跨平台桌面数据质量管理工具集。

## 功能模块

- 📊 数据校验 - Excel/CSV 格式、内容、逻辑校验
- 🖼️ 图像检测 - 批量图像质量检测（AI驱动）
- 📄 PDF处理 - 合并、拆分、加密、水印
- 🔒 MD5校验 - 文件完整性验证
- 🏷️ 元数据封装 - 文件属性管理
- 📋 文档管理 - Excel字段自动提取

## 开发

```bash
# 安装依赖
pnpm install

# 开发模式
pnpm dev

# 构建
pnpm build

# 测试
pnpm test
```

## 技术栈

- Electron 28+
- Vue 3.4+
- Vite 5+
- TypeScript 5+
- pnpm workspace (Monorepo)
